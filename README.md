# Project Squirtle Frontend

The frontend portion of the Project Squirtle website redesign. Built using React and Next.js.

## Setup

1. Clone this repo
2. Run `npm install`
3. Run `npm run dev`

## Deployment

1. Create a Heroku account if you do not have one already
2. Install the Heroku CLI
3. In the project directory, run `heroku create project-squirtle-USERNAME`, where `USERNAME` is replaced with your username
4. Make sure the changes you want to deploy are commited to git
5. Run `npm run deploy` to deploy your changes.
